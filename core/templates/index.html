About the tool Steam hunter

Current tool is my refactored cs50 final project from Harvard University. Steam hunter checks user profiles and helps to get the information on gaming activity. The tool is able to process any type of steam profile (default/custom links) and save the stats about the user in the database.

The tool is using Steam API to process the calls. If you want to keep track of your submissions, you need to create an account. Each submission aftewards will be associated with the user and available on the history page.
